¿QUE ES HTML? todo lo que debes de saber

 Tu sabes a que se refiere el HTML?

Suelen decir que es un lenguaje de programación pero solo es un marcador que nos ayuda a definir la estructura de tu sitio web. consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera. Se les conoce como etiquetas  

Etiqueta de Párrafo <p>

El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos.

Sus etiquetas son: <p> y </p> (la de cierre es opcional).

Crea una caja: en un bloque

Está definido como: Elementos de frase.

Puede contener: cero o más elementos en linea.

 

Vamos a colocar un ejemplo: Si yo quiero que la siguiente frase “ La UVM es una excelente escuela

para estudiar por sus grandes profesores que son muy preparados y te ayudan a lograr tu

prometido”

y nosotros lo queremos especificar que se trata de un párrafo se colocan las etiquetas ( <p>):

¿Como se conforman?

  • La etiqueta de apertura: consiste en el nombre del elemento (en este caso, p), encerrado por

paréntesis angulares (< >) de apertura y cierre. Establece dónde comienza o empieza a tener

efecto el elemento —en este caso, dónde es el comienzo del párrafo

2.La etiqueta de cierre: es igual que la etiqueta de apertura, excepto que incluye una barra de

cierre (/) antes del nombre de la etiqueta. Establece dónde termina el elemento —en este caso

dónde termina el párrafo-

3. El contenido: este es el contenido del elemento, que en este caso es sólo texto.

4. El elemento: la etiqueta de apertura, más la etiqueta de cierre, más el contenido equivale al

elemento.
